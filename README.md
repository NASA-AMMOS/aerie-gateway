## Develop

First make sure you have [Node.js LTS](https://nodejs.org) installed. Set the default `POSTGRES_USER` and `POSTGRES_PASSWORD` [environment variables](./docs/ENVIRONMENT.md), then do:

```sh
npm install
npm run build
npm start
```
